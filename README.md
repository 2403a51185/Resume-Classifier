# 📄 Resume Classifier using Machine Learning

## 📌 Overview
This project is a Machine Learning-based Resume Classifier that predicts job roles based on resume text.

## 🚀 Features
- Classifies resumes into different job roles
- Uses NLP techniques
- Provides decision: Select / Reject
- Displays reason for decision

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (TF-IDF Vectorizer)

## 📂 Project Structure
- RESUME_QUALITATIVE_DETECTOR_PROJECT.ipynb → Main notebook
- dataset.csv → Dataset
- resume_model.pkl → Trained ML model
- vectorizer.pkl → Text vectorizer

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Upload dataset.csv
3. Run all cells

## 📊 Output
- Predicts job role
- Gives decision (Select / Reject)
- Provides reasoning
